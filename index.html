Welcome to ExchangeME!

ExchangeME is your go-to platform for real-time currency exchange rates, historical trends, and personalized alerts. Whether you're planning a trip, managing international transactions, or simply curious about current exchange rates, we've got you covered!

Key Features

  • Real-Time Exchange Rates: Access up-to-date exchange rates for major currencies sourced from trusted financial providers.
  • Currency Conversion Tool: Convert any amount between currencies with a simple and intuitive interface.
  • Alert System: Set custom notifications for when exchange rates reach a specific level.
  • Historical Rate Data: Visualize exchange rate trends over time with interactive charts.

How It Works

Our platform allows users to select their base and target currencies, enter an amount, and instantly see the conversion value. You can also explore historical data, and set up personalized alerts to stay updated on the latest exchange rate fluctuations.

Technologies Behind the Platform

  • Fixer.io API: Provides real-time, accurate exchange rate data from trusted sources.
  • SCSS: Helps streamline the styling of the platform with enhanced flexibility and scalability.
  • Supabase: Stores historical rate data and user preferences for a personalized experience.
  • Backend Server: Handles interactions with APIs and manages data transactions.
